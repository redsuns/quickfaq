quickfaq
========
__v1.0__

QuickFaq wordpress plugin


Installing
============

Via Github
------------
Clone this plugin into your wp-content/plugin folder 
`git clone git@github.com:redsuns/quickfaq.git`


Via composer
------------
`"require": {"redsuns/quickfaq": "dev-master"}`
